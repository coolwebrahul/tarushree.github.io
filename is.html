Instance-level Segmentation

By: Tarushree Gandhi           [The Hi-tech Robotic Systemz Pvt. Ltd.]

Instance segmentation, the automatic delineation of different objects appearing in an image, is a problem within computer vision that has attracted a fair amount of attention. Segmenting at the instance level means segmenting every object.
As an extension of regular semantic segmentation, the task is to generate a binary segmentation mask for each individual object along with a semantic label. Instance segmentation is more challenging than other pixel-level learning problems such as semantic segmentation where each pixel can belong to a set of predefined groups (or classes). Whereas, in instance segmentation the number of groups (instances) is unknown a priori.
Counting and segregating the elements in an image has interest in its own right as it has a wide range of applications. With respect to street scene understanding, instance-level segmentation helps in depth estimationa nd distance calculation of different vehicles and objects present on the road.
I designed a mupltiple-branch network for simultaneous class-wise and instance-wise image segmentation. By tweaking network layers and loss function, I achieved 45% instance accuracy on Cityscapes test dataset.

Network, Framework used : Multiple-brach Encoder-Decoder network, Torch7 (Programming Language - Lua)
