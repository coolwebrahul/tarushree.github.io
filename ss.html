Semantic Segmentation

By: Tarushree Gandhi, Gaurav Pathak           [The Hi-tech Robotic Systemz Pvt. Ltd.]

For few decades image segmentation was a complex task in computer vision. Now it is more easy with deep learning. Image/semantic segmentation is understanding an image at pixel level. In other words, each pixel is assigned to a specific class or category in the image. Image segmentation successfully answers questions like : "What's in this image, and where in the image is it located?"
I used Convolutional Neural Networks to approach this problem. I implemented Enet for semantic segmentation of images into road, fourwheelers, two-wheelers and person. The model was trained extensively on Indian road data along with online datasets like Cityscapes, Berkeley Deep Drive, Mapillary Vistas, GTA etc. I incorporated Resnet features into Enet to achieve 65% accuracy on test data running at a speed of 7 fps on Nvidia TX1.

Network, Framework used : ENet, Torch7 (Programming Language - Lua)
